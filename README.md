# Topstier | Premium Digital Design & Branding Agency

Topstier is a premier digital design agency specializing in cinematic branding, custom web development, and e-commerce marketplace growth strategies. This repository contains the official high-performance, visually immersive website for Topstier.

## 🚀 Key Features

-   **Cinematic Branding:** High-end visual identity and logo design showcases.
-   **3D Visuals:** Immersive experiences powered by Three.js and React Three Fiber.
-   **Smooth Interactions:** Fluid animations and transitions using Framer Motion and Lenis scroll.
-   **Modern Tech Stack:** Built with Next.js 15 for optimal performance and SEO.
-   **Responsive Design:** Fully optimized for all devices using Tailwind CSS.
-   **Interactive Contact:** Seamless lead generation with EmailJS integration.

## 🛠️ Tech Stack

-   **Framework:** [Next.js 15](https://nextjs.org/) (App Router)
-   **Styling:** [Tailwind CSS 4](https://tailwindcss.com/)
-   **Animations:** [Framer Motion](https://www.framer.com/motion/) & [Lenis](https://lenis.darkroom.engineering/)
-   **3D Graphics:** [Three.js](https://threejs.org/) & [@react-three/fiber](https://docs.pmnd.rs/react-three-fiber/)
-   **Language:** [TypeScript](https://www.typescriptlang.org/)
-   **Deployment:** Optimized for [Vercel](https://vercel.com/)

## 📦 Getting Started

### Prerequisites

-   Node.js (Latest LTS version recommended)
-   npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/MaryamJami1/Company-web.git
   cd Company-web
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Run the development server:
   ```bash
   npm run dev
   ```

4. Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## 🏗️ Project Structure

-   `src/app`: Next.js App Router pages and layouts.
-   `src/components`: Reusable UI components (Navbar, Footer, 3D Scenes).
-   `public/`: Static assets including brand identity logos and website images.
